# Play Store Data Analysis Dashboard

This project is a Streamlit dashboard that analyzes Google Play Store apps data.

## Tasks Completed
- Task 1: Data Cleaning and Exploration
- Task 2: Category Analysis
- Task 3: Ratings Analysis
- Task 4: Size and Installs Analysis
- Task 5: Reviews vs Rating (January Filter)
- Task 6: Dual Axis Chart (Free vs Paid Apps with Time Restriction)

## Features
- Data Cleaning
- Filtering
- Grouped Bar Charts
- Dual Axis Chart
- Time-based Visualization
- Revenue Calculation
